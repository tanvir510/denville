Cara Mengaktifkan / Menggunakan Pertanyaan Rahasia Pada ID Anda

Keamanan User ID menjadi kebutuhan yang sangat penting untuk setiap pemain judi online. Faktanya, keamanan ID para member bukan saja sepenuhnya menjadi tanggung jawab pihak agen. Namun, perlu ada kerja sama dari pihak member. Jika member sendiri tidak dapat menjaga kerahasiaan ID nya sendiri, maka sebaik apapun sistem protocol yg agen jalani untuk menjaga ID para membernya. Akan tetap sia - sia.

Meskipun kini di dalam server pokerv sudah menggunakan keamanan kedua, yaitu meminta 3 digit nomor rekening terakhir anda saat melakukan login. Hal ini masih belum dapat mengamankan ID anda sepenuhnya. Bahkan MUNGKIN, orang terdekat yang tidak anda kira, dapat memiliki niat mencuri ID dan chip anda. Bukan hal mustahil bukan?

Namun Jangan Khawatir. Ikuti Saran kami yang sangat kami rekomendasikan ini, agar ID anda anti hacker / maling.

Sistem Pkv di asikqq menyediakan fitur keamanan dengan metode pertanyaan rahasia. Apa maksudnya? Seperti yang anda ketahui, jika anda ingin login kedalam akun anda. Setelah memasukan ID dan password, selanjutnya anda akan diminta untuk memasukan 3 digit nomor akhir rekening anda. Tapi jika anda mengaktifkan sistem pertanyaan rahasia. Nanti permintaan 3 digit nomor rekening akhir anda akan terganti dengan pertanyaan rahasia tersebut.

Berikut adalah cara mengaktifkan fitur pertanyaan rahasia :

  • Login kedalam ID anda
  • Klik menu PASSWORD
  • klik menu TUKAR PERTANYAAN RAHASIA
  • Pastikan kotak kecil yang ada di samping tulisan "Non aktifkan pertanyaan rahasia" dalam keadaan TIDAK di centang.
  • Pilih pertanyaan 1 yang ingin anda gunakan.
  • Masukan jawaban yang anda inginkan sebagai jawaban dari pertanyaan pertama. Huruf besar dan kecil sangat pengaruh disini. Jadi mohon diingat huruf besar kecil dari jawaban anda. Anda bisa isi dengan jawaban yang ngawur agar lebih aman. Sebagai contoh, anda memilih pertanyaan : dimanakan tempat liburan favorite anda saat anda kecil? Anda boleh jawab : Surga (hanya contoh)
  • Pilih pertanyaan rahasia ke 2 agar jauh lebih aman.
  • Isi jawaban anda untuk pertanyaan ke 2, jangan samakan dengan jawaban dari pertanyaan nomor 1.
  • Klik OK. dan selesai, kini ID anda sangat aman selama anda tidak memberitahukannya kepada orang lain.
  • Jangan lupakan jawaban dari pertanyaan rahasia ini. buat jawaban yang ngawur, namun mudah diingat. Jika perlu catat di Handphone anda untuk jawaban atas pertanyaan rahasia yang anda pilih.

Apa Keuntungan Dari Fitur Pertanyaan Rahasia Ini ?

Tentu saja sangat menguntungkan untuk keamanan ID anda sendiri. Tidak ada pihak manapun yang ingin ID nya dicuri oleh hacker / maling, termasuk kami para agent. Kami tidak sangat tidak menginginkan ID para member kami dihack dengan orang yang tidak bertanggung jawab. Maka dari itu kami memberikan info tentang cara mengamankan ID anda sendiri.

Bedanya pertanyaan rahasia dengan permintaan 3 digit rekening akhir anda ini cukup berbeda jauh. Terbukti pertanyaan rahasia jauh lebih efisien dalam melindungi ID anda.

Kita semua tidak tau tekhnologi seperti apa yang dimiliki para hacker untuk mengambil info login para pemain judi. Apalagi tidak semua pemain judi online paham terhadap tekhnologi. Dengan menggunakan sistem pertanyaan rahasia ini, kami yakin ID anda akan jauh lebih aman jika anda tetap menjaga kerahasiaan atas jawabannya. Dan perlu diingat, mohon gunakan jawaban yang mudah diingat dan sedikit ngawur. Jika perlu kombinasi antara huruf dan angka.
